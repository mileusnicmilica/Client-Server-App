**PROJECT DESCRIPTION**

This project represents a client-server application developed in Java, designed for managing travel data. The application allows users to register, log in, and manage travel records.

**Project Structure**

The project is organized into three main modules:

Klijent – graphical user interface (forms for login, registration, and travel management)

Server – handles client requests and business logic

Zajednicki – shared classes used by both client and server

**Technologies Used**

Java

Socket Programming

Java Swing (GUI)

(add database if used, MySQL)

**How to Run**

Start the Server application

Run the Klijent application

Log in or register a new user

**Features**

User registration

User login

Add travel records

View travel records
